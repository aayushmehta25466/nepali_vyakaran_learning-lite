# Week 5 Assignment - Lightweight Web Content

Interactive learning platform built with pure HTML, CSS, and JavaScript. Optimized for performance and bilingual support (English/Nepali).

## Files

```
week5-assignment/
├── index.html           # Main HTML file
├── styles.css           # Styling
├── script.js            # Interactivity and language switching
├── DOCUMENTATION.md     # Optimization details
├── TEST_CASES.md        # Test cases
└── README.md            # This file
```

## Quick Start

1. Open `index.html` in any modern browser
2. No dependencies or build process required

## Key Features

- **Lightweight**: 25KB total size (90% smaller than React version)
- **Bilingual**: English (default) and Nepali language support
- **Performance**: ~2s load time on 3G networks
- **Responsive**: Mobile-first design with adaptive layouts
- **Accessible**: WCAG AA compliant with keyboard navigation
